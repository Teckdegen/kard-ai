# kard-ai

> Trust infrastructure for autonomous economies.

The coordination and settlement SDK that enables autonomous agents to transact trustlessly onchain.

## Install

```bash
npm install kard-ai
```

## Full documentation

See the [docs site](https://kard-ai.vercel.app) for complete usage guides, API reference, and deployment instructions.

## Quick usage

```ts
import { Kard } from "kard-ai";

const kard = await Kard.fromEnv();
```

See `.env.example` for all required configuration variables.
